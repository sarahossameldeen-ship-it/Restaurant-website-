# Restaurant-website-
A full-stack restaurant web application designed to simulate great experience for a fancy restaurant. The frontend provides an User-friendly interface toto effectively showcase the restaurant’s quality and it's menu, while the backend manages user authentication, database operations and reservation process.
## Features
- User-friendly website with smooth interface
- Responsive design
- Interactive navigation bar to move smoothly between pages
- Reservation system to reserve a table
  ## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
  ## How to Run
1. Open the project folder
2. Run using XAMPP / local server
3. Open in browser:
   http://localhost/Resturant website/home
